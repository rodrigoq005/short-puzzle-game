Project Overview:

    *At least 4 rooms/scenes
    *Several unique interactions
    *An autoload script that tracks 1-2 variables across different scenes

  *To pick up a key, and unlock the final door. To do this, the player must enter the left and right
  *Puzzle rooms and complete them, where they will be able to pick up 10 coins for the key.
